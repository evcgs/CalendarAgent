# 日历管理智能体 - 智能时间管理助手 V1.0
## 🎯 项目说明
CalendarAgent是专门负责日历和时间管理的智能助手，支持日程安排、会议预约、智能提醒、日历同步、冲突检测等功能，帮助用户高效管理时间，提升工作效率。

### 解决的核心痛点
- ❌ 日程安排混乱，经常错过会议和重要事项
- ❌ 会议预约麻烦，需要反复确认参会人时间
- ❌ 提醒不及时，经常忘记重要 deadline
- ❌ 多设备日历不同步，信息不一致
- ❌ 日程冲突无法及时发现，导致时间浪费

## ✨ 核心特性
### 📅 日程管理
- ✅ 智能日程创建：自然语言描述即可创建日程，自动识别时间、地点、参会人
- ✅ 日程查询：按日期、关键词、类型查询日程，生成时间安排表
- ✅ 日程修改：自动调整日程时间，批量修改重复日程
- ✅ 智能冲突检测：创建日程时自动检测时间冲突，给出调整建议

### 🤝 会议管理
- ✅ 智能会议预约：自动查询参会人空闲时间，推荐最佳会议时间
- ✅ 会议邀请：自动发送会议邀请，同步到参会人日历
- ✅ 会议提醒：会前自动提醒，发送会议议程和材料链接
- ✅ 会议纪要：自动生成会议纪要，同步行动项和截止时间

### 🔔 智能提醒
- ✅ 多渠道提醒：支持飞书、微信、短信、邮件等多种提醒方式
- ✅ 自定义提醒规则：可设置提前5分钟、15分钟、1小时等多轮提醒
- ✅ 重要事项优先级提醒：高优先级事项强提醒，避免错过
- ✅ 重复性提醒：支持每日、每周、每月等周期性提醒

### 🔄 多平台同步
- ✅ 多日历同步：支持飞书日历、Google日历、Outlook日历同步
- ✅ 跨设备同步：手机、电脑、平板多端实时同步
- ✅ 团队日历共享：支持团队成员日历共享，方便安排协作时间
- ✅ 节假日自动识别：自动识别国内法定节假日和调休安排

## 🚀 安装方式
### 方式1：OpenClaw技能市场安装（推荐）
1. 打开OpenClaw控制面板 → 技能市场
2. 搜索「日历管理智能体」
3. 点击「安装」即可自动完成部署

### 方式2：本地安装
```bash
# 1. 下载到Agent目录
cd ~/.openclaw/agents
git clone https://github.com/evcgs/calendar-agent.git

# 2. 安装依赖
cd calendar-agent
npm install
```

### 方式3：Docker一键部署
```bash
docker run -d \
  --name calendar-agent \
  -v ~/.openclaw/calendar-agent-config:/app/config \
  evcgs/calendar-agent:v1.0.0
```

## 📝 使用指南
### 基础用法
直接用自然语言描述需求即可：
```
"明天下午2点安排一个项目评审会，参会人：产品、研发、测试，时长2小时"
"查看本周所有的会议安排"
"明天上午10点提醒我提交周报"
"把下周三的产品发布会改到下周四下午3点"
"查询下周有没有空闲的全天时间，安排团建"
```

### 高级用法
可以指定特殊要求：
```
"安排下周一的周会，每周重复，提醒时间提前15分钟，所有参会人都会收到飞书提醒"
"导出本月的日程统计，按照会议类型分类，生成时间占比报表"
"同步我的Google日历到飞书日历，保留所有未来3个月的日程"
```

## 📋 典型使用场景
### 1. 个人时间管理
> 智能安排个人日程，避免遗漏重要事项
```
"帮我安排下周的学习计划，每天晚上8点到10点学习AI相关知识，周末下午运动"
```

### 2. 团队会议安排
> 自动查找参会人共同空闲时间，高效安排会议
```
"安排一个项目需求评审会，参会人有产品经理、前端开发、后端开发、测试，找一个大家都有空的时间，下周三之前"
```

### 3. 重要事项提醒
> 多渠道提醒重要事项，确保不会错过
```
"3月15日是报税截止日期，提前3天每天提醒我，提醒方式是飞书+短信"
```

### 4. 时间统计分析
> 统计时间使用情况，优化时间分配
```
"统计我上个月的时间使用情况，看看开会占了多少时间，有多少时间是在处理邮件"
```
